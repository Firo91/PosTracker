Installation Steps
  1. Download the ZIP package above
  2. Extract to a folder on the target Windows machine (e.g., C:\NetWatch\agent)
  3. Right-click INSTALL_AGENT.bat and select "Run as Administrator"
  4. The agent will be installed as a scheduled task running every minute
  5. Check status by running CHECK_STATUS.bat
Requirements: Windows Server or Windows 10/11 with PowerShell 5.1+
